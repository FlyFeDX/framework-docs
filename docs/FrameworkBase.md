# 框架基础能力

- 完全响应式布局（支持电脑、平板、手机等所有主流设备）
- 强大的一键生成功能（包括控制器、模型、视图、菜单等）
- 支持按钮及数据权限，可自定义部门数据权限
- 国际化支持，服务端及客户端支持
- 完善的日志记录体系简单注解即可实现
- 支持服务监控，数据监控功能
- 支持多应用热插拔，应用间相互嵌套使用
- 支持主微应用数据消息共享，应用间相互通信
- 支持动态切换主题，一键换肤
